## Features

- Dashboard allows user to search city of their choice
- Keeps search history
- Shows weather icon such as rain, sunshine etc
- Today's weather
- Five day forecast 
- Weather shows: 
    - Temperature
    - Wind Speed
    - Humidity
    - UV Index
